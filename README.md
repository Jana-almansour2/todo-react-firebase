# Todo App with React & Firebase

A simple Todo app built with **React** and **Firebase**.  
Users can login with Google, add, delete, and view only their own tasks.

## Features

- Login with Google account
- User stays logged in until logout
- Each user sees only their own todos
- Add and delete todos
- Simple, responsive UI
- Firebase Firestore used for database
- Authentication handled via Firebase Auth

## Technologies Used

- React
- Firebase (Auth + Firestore)
- CSS for styling
